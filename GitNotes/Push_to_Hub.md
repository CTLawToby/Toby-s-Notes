<h1>Simple notes about how to push new change to github</h1>
<ol>
    <li>
        Save the file you created or edited
    </li>
    <li>
        <code>git add .</code>
    </li>
    <li>
        <code>git commit -m "<em>message</em>"</code>
    </li>
    <li>
        <code>git push</code> 
    </li>
</ol>
